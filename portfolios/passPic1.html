Password Generator 5000:
This is a simple password generator that will ask you for inputs as to how you would like your password to be generated. How we solved the Issue Using simple javascript alert boxes you will be asked If you would like to include lowercase letters If you would like to include uppercase letters If you would like it to include numbers If you would like it to include special characters & finally the length of the password, anywhere from 8-128 characters (note: if a number lesser than 8 or greater than 128, user will be prompted to try again) Finally outputting a password that meets the above specified criteria
